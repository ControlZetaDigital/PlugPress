# Descripción

Un plugin compuesto por pequeños programas llamados plugs que permiten una sencilla y 
rápida configuración para que desarrolladores puedan personalizar sus sitios creados
en Wordpress de forma rápida, personalizada y sin necesidad de instalar numerosos 
plugins que ralenticen la carga.
